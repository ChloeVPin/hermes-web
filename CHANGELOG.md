## [1.0.0] 2026-04-24

![status](https://img.shields.io/badge/release-stable-ffac02?style=flat-square&labelColor=170d02)

Initial release of Hermes-Web.

### Added

- Browser UI for Hermes Agent
- Streaming chat with message deltas, thinking, and reasoning display
- Session management and chat history
- Skills, tools, and slash command browser
- Interactive approval and clarify flows
- Rust bridge and Python bridge support
- Cross-platform installers and launch helpers
- Documentation and release metadata cleanup
