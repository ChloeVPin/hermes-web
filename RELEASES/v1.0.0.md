# Hermes-Web v1.0.0

Initial release of Hermes-Web.

## Highlights

- Browser UI for Hermes Agent
- WebSocket bridge to the Hermes Agent TUI gateway
- Streaming chat with message deltas
- Session management and history
- Tool, skill, and slash command browsing
- Approval, clarify, sudo, and secret prompts
- Rust bridge and Python bridge support
- Install scripts for Linux, macOS, WSL, Termux, and Windows
